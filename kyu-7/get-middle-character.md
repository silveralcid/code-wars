### Task
Given a **non-empty string**, return the **middle character(s)** of the string.

- If the string length is **odd**, return the **middle character**.  
- If the string length is **even**, return the **middle two characters**.

---

### Examples

`"test"` → `"es"`  
`"testing"` → `"t"`  
`"middle"` → `"dd"`  
`"A"` → `"A"`
